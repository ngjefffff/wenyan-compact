# 简写规则

**类型标签**（写在 ## 后，可多标）
```
定   决策已拍板      弃   路径已关闭
为   对世界有改变    知   新事实 / 新定义
疑   未决张力        续   下一步方向
演   推理演变轨迹（起承转合，选用）
```

**逻辑符号层**
```
→      因果 / 导向
∵ ∴    因为 / 所以
⊕      排除 / 互斥
>      优先级
弃X取Y  标准排除路径格式
[t1]   时序标记，链长 >4 步时启用
```

**现代格式保留区**
文件路径 · session ID · commit hash · 版本号 · branch 名
嵌入文言中不压缩，不翻译。

**质量标准**
- 一简一命题，抽出来单独读仍可激活
- 排除路径（弃X：因Y）不可省略，不可折叠进结论
- 理由重于结论：`取X` 不如 `弃Y取X，因Z`
- 密度优先：能四字格不用一句话，能用符号不用文字
